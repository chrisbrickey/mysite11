My Story

I used to strategize about products and markets; now I build them.

  • User Experience & Strategic Programs

    From improving healthcare experience through collaboration with biomedical engineers to producing marketing campaigns in NYC, I focused on understanding customer needs and using technology to fill knowledge gaps. I continued this mission when running strategic programs for a global, medical device manufacturer. I often built solutions myself, including a consumption model to forecast product in the field, and included software engineers to assist in root cause analyses.

  • Strategy Consulting Across 3 Continents

    Branching out by industry and geography, I led consulting teams for large corporate clients in tech, e-commerce, and retail - to name a few. Notable projects include data strategy, CX, valuation, and portfolio optimization. For one of the largest tech acquisitions of all time ($16B), I developed a research-based global co-branding strategy, which retained revenue during the transition and later boosted market share with co-branded products.

  • Transition to Software Development

    Inspired by the opportunities that software makes possible, I taught myself to code in a few languages. Then I couldn't stop. I was up all night building games and problem-solving algorithms. That didn't satiate my curiosity so I moved on to constructing small programs like this website and a shopping cart web app. Though I still deploy customer research and strategic planning to guide projects, I now prefer to develop software full-time.

  • Software Engineering Apprentice

    Getting to learn every day is the best! At Ragtag, I work on a team with more senior engineers to build components of an API that is a national Call-To-Action aggregator. I'm also a full-time student at App Academy, the elite programming immersive in San Francisco, where I work intensely to round out my skillset across front-end and back-end. I continue to build personal projects and will be available for a full-time gig in August 2017.

Shop the World

an e-commerce rails app for two-sided markets

This Rails app allows administrators to populate a product catalog. Shoppers can view and purchase those items. This app is product agnostic. The catalog can be populated with anything, including these tangine ceramics at a Marrakech souk. I visited there a few years ago. The pottery is beautiful, but I'd rather order the delicious meals that were cooked and served in them!

The files for this program are available at my github repository.

Space Race Game

a puzzle-solving and choose-your-own adventure game written in python

I'm obsessed with space travel and revelled in writing a game about it. Players load the game from the command line and are prompted through a story. The story changes based on the user's input. Players must solve games of logic and chance in order to avoid a pre-mature GAME OVER.

The files for this program are available at my github repository.

Scraping for Progress

scrapes URLs and parses the event content for input into APIs

This webscraper incorporates the logic to parse data from multiple target URLs. It uses Ruby and the Nokogiri gem, among others. The outputs are structured as JSON objects so they can directly input to APIs. The targets URLs are currently websites of politically progressive organizations in the US. The purpose is to cull data on call-to-action events and feed the data directly into aggregator APIs so that those events can be accessed more efficiently by community organizers on a global scale.

The files for this program are available at my github repository.

GoTango

UX design and product management for a tango instruction app

I worked as both product manager and UX designer on this project. Learning tango is an ambitious undertaking, as I discovered when I travelled to Buenos Aires for that purpose. Dancing tango requires lots of practice, but the most common forms of practice - milongas and group classes - are often inconvenient and intimidating. GoTango uses mobile technology to make tango practice more easily a part of one's lifestyle - more spontaneous.

The files for this project are available at my github repository.
