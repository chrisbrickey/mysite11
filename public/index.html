My Story

I used to strategize about products and markets; now I build them.

  • User Experience & Strategic Programs

    Early in my career, I focused on using technology to meet customer needs. For a global manufacturer, I investigated and modeled issues with medical devices to determine root cause and propose solutions. In collaboration with biomedical engineers, I implemented process manufacturing changes, new product development, and strategic ventures to improve safety and customer experience.

  • Strategy Consulting Across 3 Continents

    For over 4 years, I led consulting teams for large corporate clients in tech, biotech, and media - to name a few. For a major tech aquisition, I developed a research-based global brand strategy, which retained the customer base during transition. Notable projects include CX, emerging market entry, valuation, portfolio optimization, and data strategy.

  • Transition to Software Development

    I taught myself to code (first in Python) and quickly became addicted - staying up all night building games and small apps. I expanded my skillset by pair programming with senior developers, studying CS theory, and building more sophisticated web apps with Django, Rails, and React/Redux frameworks. At Ragtag, I built features for an open-source Rails API that is a national Call-To-Action aggregator.

  • Engineering at Square and CashApp

    In 2018 I began working for Block, Inc. as a software engineer. I led engineering projects including extraction of a service from the monolith. I adapted the banking and investment platform to support the launch of Square's new U.S. bank. And, for the global consumer-facing CashApp, I worked across highly distributed microservices to architect and optimize a paradigm shift in ledgering to support new products.

flashcards4u

Java SpringBoot flashcard app that pulls content from csv source files

I use flashcards to learn new natural and programming languages, but I hate writing out and carrying around physical flashcards. So I made this app to automate and host flashcards on the web. Future features include filtering by category and source file.

Live link: http://www.flashcards4u.com

Source code: my github repository

insta-étoile

Rails / React / Redux media-sharing app for performing artists, includes drag/drop upload

Danseur Étoile (literally 'star dancer' in French) is the highest rank within the Paris Opera Ballet. insta-étoile is a visual platform for performing artists of all kinds to instantly share and showcase photos of their craft while building a community of supporters and followers. insta-étoile was an exercise for me in mimicking the look, feel, and functionality of Instagram as closely as possible.

Live link: http://www.insta-étoile.com

Source code: my github repository

Find CokeZero API

Django REST Framework API for finding stores that carry your favorite flavors of Coca-cola

Cherry CokeZero was - at one time - my beverage of choice. When Coco-Cola changed the flavor formula of Cherry CokeZero, I panicked! Soda inventories vary from store to store so - before findcokezero.com - I lacked a reliable way to find the original Cherry CokeZero formula. This test-driven, hyper-linked, browsable API allows users to quickly find, add, and update retail locations that carry the different flavors and formulas of Coca-Cola products. Future development plans include a map-based UI (see github for wireframes).

Live link to landing page: http://www.findcokezero.com

Live link to browsable API: http://www.findcokezero.com/api

Source code: my github repository
