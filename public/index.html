My Story

I used to strategize about products and markets; now I build them.

  • User Experience & Strategic Programs

    Early in my career, I focused on using technology to meet customer needs. For a global manufacturer, I investigated and modeled issues with medical devices to determine root cause and propose solutions. In collaboration with biomedical engineers, I implemented process manufacturing changes, new product development, and strategic ventures to improve safety and customer experience.

  • Strategy Consulting Across 3 Continents

    For over 4 years, I led consulting teams for large corporate clients in tech, biotech, and media - to name a few. For a major tech aquisition, I developed a research-based global brand strategy, which retained the customer base during transition. Notable projects include CX, emerging market entry, valuation, portfolio optimization, and data strategy.

  • Transition to Software Development

    I taught myself to code (first in Python) and quickly became addicted - staying up all night building games and small apps. I expanded my skillset by pair programming with senior developers, studying CS theory, and building more sophisticated web apps with Django, Rails, and React/Redux frameworks. At Ragtag, I built features for an open-source Rails API that is a national Call-To-Action aggregator.

  • Engineering at Square and CashApp

    Since 2018, I've been a software engineer at Block, Inc. where I led engineering projects including extraction of reporting functionality to a microservice and adapting the platform to support Square's newly launched bank. I designed and implemented the services API for a greenfield project, which we launched within 3 months! I currently work across distributed services supporting the consumer-facing CashApp.

insta-étoile

Rails 5 / React / Redux media-sharing app for performing artists, includes drag/drop upload

Danseur Étoile (literally 'star dancer' in French) is the highest rank within the Paris Opera Ballet. insta-étoile provides a visual platform to enable performing artists of all kinds to instantly share and showcase photos of their craft while building a community of supporters and followers. insta-étoile will further the careers of performing artists by broadcasting their experience in a beautiful and digestible format, linking robust immersive profiles with an artist's community and following.

Live link: www.insta-étoile.com

The files for this program are available at my github repository

Find CokeZero API

Django REST Framework API for finding local stores that carry your favorite flavors of Coca-cola

Cherry CokeZero is my comfort food, but soda inventories vary so - before findcokezero.com - there was not a reliable way to find it. This test-driven, hyper-linked, browsable API allows users to quickly find, add. and update retail locations that carry the different flavors and formulas of Coca-Cola products. Future development plans include a map-based UI (see github for wireframes).

Live link: www.findcokezero.com

The files for this program are available at my github repository

Firework-Fun

Vanilla JavaScript activity with custom trigonometric animations

This activity is set against a night sky and allows users to design their own fireworks show by choosing the location, color, and relative timing of fireworks. The explosion trajectories of the fireworks are generated by custom trigonometric algorithms.

Live link: www.firework-fun.com

The files for this program are available at my github repository

Equity-Portfolio

Django stock portfolio app with live update from 3rd party service, includes browsable API

This web app allows non-technical users to search for equities across multiple markets and include them in their portfolio with amount of shares owned. The portfolio updates so users can view the current value of each investment. This app utilizes Django for both front and back end as well as Django Rest Framework to provide a browsable API for technical users.

Live link: www.equity-portfolio.com

The files for this program are available at my github repository

GoTango

UX design and product management for a tango instruction app

I worked as both product manager and UX designer on this project. Learning tango is an ambitious undertaking, as I discovered when I travelled to Buenos Aires for that purpose. Dancing tango requires lots of practice, but the most common forms of practice - milongas and group classes - are often inconvenient and intimidating. GoTango uses mobile technology to make tango practice more easily a part of one's lifestyle - more spontaneous.
